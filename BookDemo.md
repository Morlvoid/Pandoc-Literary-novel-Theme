# 序言：考古学的铲子

这部书并非颂神的圣典，亦非弑神的檄文。它是一把插进时光淤土的铲——试图从金箔的谎言、血书的战栗、树皮的余温中，筛出被神性碾碎的尘世真相。

书中记载了约2400年前太阳神与月亮神的故事。四个主要人物：阿贝斯（Abyssos）、卡兰纱（Calantha）、伊莉丝（Liris）、禾洛（Herōas）。

我们不做弥合者，只当裂缝的拓印匠。

# 第一章 · 从人到神

## 1.1 弃儿泰奥菲卢斯

他原名泰奥菲卢斯，意为“被神所爱者”。

童年陶片上，他曾是偷摘葡萄的赤脚少年。流亡途中，母亲赠他铜勺，愿此勺丈量一生温饱。敌国诅咒泥板与难民绳结，记录了城邦陷落、千人流亡的残酷。

> “母亲咳血浸透我后背……真可笑，被神所爱的孩子，在吃死人粮。”
> ——流亡者裹尸布日记

## 1.2 成神

十六岁完成“十三重任务”，二十九岁更名阿贝斯（Abyssos），意为“深渊”。

他降临恋人卡兰纱的故国，以神力缔造和平。废奴、铺路、降下麦穗雨——乌托邦真实存在过。

| 时代     | 特征           | 考据来源           |
| :------- | :------------- | :----------------- |
| 成神初期 | 理想主义与权能 | 金碑、更名诏书     |
| 神代中期 | 疯魔的永生牢笼 | 酒商账本、骨铃     |
| 神代晚期 | 献祭与幻灭     | 树皮契约、水晶鳞片 |

# 第二章 · 月亮的囚笼

## 2.1 卡兰纱：被神化的死者

卡兰纱，月亮神，战死时仅十八岁。

阿贝斯将她塑造成“月光战神”，但真相是：她曾夜闯敌营救孩童，是反战者。阿贝斯用两千年建造画廊，用敌军铠甲、囚徒镣铐为颜料，反复重绘她的肖像。

> “我们画的究竟是女神，还是他绝望的倒影？”
> ——神像画师日记

## 2.2 伊莉丝：替身的清醒

伊莉丝，第二代月神，与卡兰纱容貌相同。她被囚禁、被灌药、被要求复刻亡者的声线。

但她布下陷阱：将卡兰纱的真品月光石缝入禾洛衣襟，将逃生路线刻进陶土耳钉。

> “白裙子今早咬断了舌头。血滴在我衣上像凋谢的石榴花。”
> ——侍女染血衣襟

# 第三章 · 凡人的微光

## 3.1 白发少年禾洛

禾洛，天生白发，被村民视为恶魔。他祈求太阳神，却只看到冷漠。直到某个夜晚，月亮神的目光垂落于他。

> “我渴求她的垂怜。这念头大逆不道，但我无法抑制。”

十八岁生日前，他坠入湖中，被月光所救，获得奇异力量。

## 3.2 献祭

太阳神濒死，国度崩塌。禾洛闯入神殿，看到月亮神抱着垂死的阿贝斯。他跪下来，握住了太阳神布满皱纹的手。

古老的咒语从他唇间流出。太阳神重获生机，而他的生命只剩最后一天。

## 3.3 树皮书

他在五片树皮上刻下遗言，符号与碎片拼凑出真相：

| 残片 | 刻痕                         | 解读               |
| :--- | :--------------------------- | :----------------- |
| 01   | 月牙 白                      | 自我标识与信仰     |
| 02   | 湖 月牙 光/暖                | 坠湖获救           |
| 03   | 月牙 村乱 神宫               | 末世与神殿         |
| 04   | 白神 老阳 跪 握 流           | 仪式核心：神力置换 |
| 05   | 爱月 愿安 值 角落 ...月照... | 临终心迹           |

> “值”——此字刻痕深稳，与“握”之挣扎对比鲜明。

他蜷缩在角落，在月光中停止呼吸。无人知晓。

# 尾声：光从裂缝中升起

阿贝斯用两千四百年搭建时间牢笼，囚禁的从不是月亮，而是那个跪在雨夜里攥紧染血羊毛腕带的少年泰奥菲卢斯。

禾洛临终的“月照...”，是对伊莉丝的应答：“光在人间。”

他早已悟透：真实的神性，存于凡人直面悲剧时仍愿捧出的那寸温热。

众神已溺毙于自身命名的深渊。而尘世的光，正从所有裂缝中升起。
